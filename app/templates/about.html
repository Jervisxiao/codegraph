About the website

This is an open-source tool that helps visualize your PYTHON code logics and structures.

Notes:

1. Click "Home" on the top right corner or "An Open-Source Code Visualization Tool" to enter the main page.

2. You could put your github link in the InputField.

3. The link should start with "https://raw.githubusercontent.com/".

4. Click "Visualize" button.

5. A tree diagram will show up and display your code logics and structures
